Figure role:
This is Figure 2, the overall framework of SkillWiki. It should explain the full system, not just motivation.

Canvas:
- Wide figure* layout, around 0.95 textwidth by 3.5 inches.
- Aspect ratio about 2.6:1 to 3:1.
- Pure white background.
- No figure title outside the diagram, no caption, no black outer frame.
- Fill the canvas tightly with small margins.

Style:
- Match the friendly hand-drawn academic cartoon style of Figure 1 v7.
- Rounded sketch-like panels, soft muted colors, simple icons, tidy arrows.
- Neat hand-drawn readable labels, not too childish and not corporate.
- Palette: muted teal, cobalt, lavender, emerald, amber, coral, soft gray.
- All text horizontal.
- Keep labels short and readable.

Core story:
SkillWiki is an autonomous skill production and governance infrastructure.
It has two coupled autonomous agent frameworks:
1. Task Execution Agents use released skills to solve tasks and produce execution feedback.
2. Self-Management Agents build, audit, monitor, maintain, and evolve the skill ecosystem.

High-level layout:
Use a left-to-right pipeline with a feedback loop.

Left column: "Knowledge Sources"
Show source icons/cards:
- trajectories
- documents
- API specs
- scripts
- past skills
- agent executions

Middle-left panel: "Knowledge-Grounded Skill Construction"
Show a production line:
extract actions -> normalize -> summarize -> build candidate
Use small agent icons around the line.

Center panel: "SkillWiki"
This is the central repository/wiki hub.
Inside or around it show:
- Skill assets
- Skill schema
- Provenance graph
- Lifecycle states
Use small labels:
S1 Candidate, S2 Draft, S3 Verified, S4 Released

Top lane: "Task Execution Agents"
Show:
retriever -> planner -> executor -> verifier -> reflection
They read from released Skill assets and send task results / execution experience back.

Bottom lane: "Self-Management Agents"
Show:
builder -> auditor -> monitor -> maintainer -> librarian
They generate proposals, snapshots, diffs, releases, and repaired skill versions.

Right panel: "Governed Skill Ecosystem"
Show organized skill graph and versioned skill cards.

- snapshot / diff / release
- maintenance proposal
- versioned skills

Feedback loop:
